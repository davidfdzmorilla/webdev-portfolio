A fully self-contained knowledge base platform that processes text, code, and markdown using deterministic NLP techniques — no external AI APIs required.

## Features

- **TF-IDF Keyword Extraction**: Identifies key terms using term frequency analysis
- **Full-Text Search**: SQLite FTS5 with highlighted snippet extraction
- **Auto Language Detection**: TypeScript, JavaScript, Python, Go, Rust, Markdown, JSON, YAML
- **Rule-Based Auto-Tagging**: testing, api, database, auth, devops, react, async, error-handling
- **Q&A Simulation**: Finds the most relevant sentence chunk for any query
- **Statistics Dashboard**: Language breakdown, keyword cloud, reading times
